API Information

The APIs provided by this application are as follows:

  • GET /subscribers: Fetches a list of all subscribers from the database.
  • GET /subscribers/names: Retrieves an array containing the names and subscribed channels of subscribers.
  • GET /subscribers/:id: Provides details of a subscriber identified by the given ID.
